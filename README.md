# ConCalc Mobile

ConCalc is a React Native + Expo mobile app for construction material estimates and starter local building-code reference notes.

## Features

- Concrete slab calculator
- Drywall sheet calculator
- Flooring square-foot calculator
- Framing studs and plates calculator
- Local code lookup starter data

## Test on Android with Expo Go

1. Install Node.js on your computer.
2. Install Expo Go on your Android phone from Google Play.
3. Clone this branch:

```bash
git clone -b react-native-version https://github.com/nilszachca/ConCalc.git
cd ConCalc
npm install
npx expo start
```

4. Scan the QR code with Expo Go.

## Important code note

Building code data in this app is starter reference data only. Always verify requirements with your local Authority Having Jurisdiction.
